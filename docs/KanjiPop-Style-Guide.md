## Mascot

### Name
**Poppi**

### Concept
Poppi is a **small, bouncy ink droplet** who learns kanji alongside the user.

### Visual Characteristics
- Shape: Rounded teardrop / ink blob
- Eyes: Large, friendly, expressive
- Expression: Supportive and curious
- Style: Soft, simple, scalable
- Optional accessory: Small headband (study dojo vibe)

### Personality
Poppi:
- Celebrates correct answers
- Encourages effort
- Never shows disappointment
- Is a study partner, not a teacher

### Example Reactions
| Situation | Poppi Reaction |
|----------|----------------|
| Card appears | Pops into view |
| Easy recall | Big happy bounce |
| Hard recall | Determined smile |
| Wrong answer | Encouraging nod |
